# Shoes App

**Shoes App** is a platform where users can view and purchase various types of shoes. The app is designed with a user-friendly interface and developed using modern web technologies.

## Features
- Browse and purchase different shoe products.
- Smooth and responsive user interface.
- Designed with clean and minimalistic styling.

## Technologies and Libraries
The project is built using the following technologies and libraries:
- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **React Router Dom**: For managing navigation and routing within the app.
- **Tailwind CSS**: For styling and ensuring responsiveness.
- **Axios**: For making API requests.
- **JSON-Server**: To simulate a backend server for product data.
- **@tanstack/react-query**: For data fetching and state management.
- **React-Toastify**: For displaying notifications and alerts.
